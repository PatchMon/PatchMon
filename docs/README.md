# PatchMon Documentation

Simple markdown-based documentation for GitHub Pages.

## Structure

- All documentation is in `.md` files
- `index.html` provides the sidebar navigation
- GitHub Pages will serve the HTML files directly (no Jekyll)

## Adding Content

1. Create a `.md` file in the appropriate directory
2. Add a link to it in `index.html` sidebar
3. The markdown will be automatically rendered when the page loads

## Local Testing

Just open `index.html` in a browser. The markdown files will be loaded and rendered automatically.
